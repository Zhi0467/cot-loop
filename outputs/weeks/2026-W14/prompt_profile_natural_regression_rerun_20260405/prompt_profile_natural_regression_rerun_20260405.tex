\section{Metric Definitions}

\begin{itemize}
\item Regression score:
  each held-out prompt gets one scalar score
  $s_i = \texttt{predicted mean\_relative\_length}$.
\item Actual mass:
  each held-out prompt contributes
  $y_i = \texttt{actual mean\_relative\_length}$.
\item \texttt{top\_10p\_capture} / \texttt{top\_20p\_capture}:
  sort held-out prompts by descending $s_i$, keep the top 10\% or 20\%, and
  report $\sum_{i \in \mathrm{kept}} y_i / \sum_j y_j$.
\item ``Mass'' means total realized relative-length weight across held-out
  prompts, not a prompt count and not a binary positive count.
\item \texttt{RMSE}:
  pointwise error on aligned held-out prompt pairs.
\item \texttt{Spearman}:
  monotone-order diagnostic on those same aligned held-out prompt pairs. It is
  not the task definition.
\item Metadata baseline:
  train-fit prompt-only linear regression on \texttt{prompt\_token\_count} and
  \texttt{effective\_max\_tokens}. Because \texttt{effective\_max\_tokens} is
  fixed at \texttt{30000}, prompt length is the only nontrivial feature here.
\end{itemize}

\section{Interpretation}

\begin{itemize}
\item Operational screening read:
  on \texttt{top\_20p\_capture}, ensemble beats prompt length on
  \texttt{GPQA}, \texttt{MMLU-Pro}, and \texttt{LiveCodeBench}; it loses on
  \texttt{AIME} and \texttt{MATH-500}. It beats \texttt{last\_layer} on
  \texttt{4 / 5}, with only \texttt{AIME} favoring \texttt{last\_layer}.
\item Calibration read:
  on \texttt{RMSE}, ensemble beats prompt length only on \texttt{GPQA} and
  \texttt{LiveCodeBench}. Prompt length still wins on \texttt{AIME},
  \texttt{MATH-500}, and \texttt{MMLU-Pro}.
\item Diagnostic ordering:
  \texttt{Spearman} still looks strongest for ensemble on
  \texttt{GPQA} and \texttt{LiveCodeBench}, but it remains tertiary and should
  not drive the headline claim.
\item Most important rerun result:
  the mixed regression story did not wash out, sharpen, or drift on the current
  branch. It reproduced exactly.
\end{itemize}

\section{Recommendation}

\begin{itemize}

\item Keep \texttt{top\_20p\_capture} as the primary regression metric.

\item Keep \texttt{Spearman} diagnostic only.

\end{itemize}
